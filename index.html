① 기본 흐름
탭 전환상단 테바·테메·토바·토메·저그중립·혼종 탭으로 카테고리를 전환합니다. 탭에 선택된 유닛 수가 뱃지로 표시됩니다.
전체 선택탭 상단 전체 선택 (N개) 버튼으로 히든·슈퍼히든·레전드를 일괄 선택/해제합니다.
유닛 선택유닛 행(이름+조합조건)을 클릭하면 선택됨 구역으로 이동합니다. 다시 클릭 → 해제.
수량 조절각 행 오른쪽 − 수량 +으로 1~8 조절. 매직코스트·정수 모두 배수 반영됩니다.
정수확보히든·슈퍼히든 선택 시 정수확보 뱃지가 자동으로 표시되고 정수 합산에 반영됩니다 (유닛당 1개 기준).
② 매직코스트 — COST 패널
상단 5칸총 매직코스트·총 정수합산·코랄·아이어·제루스 정수. 녹색 배경이 정수 관련입니다.
하단 20칸하늘색 배경이 매직 유닛 슬롯입니다. 활성화 시 해당 유닛 필요 수량이 표시됩니다.
💡 cost 필드 없는 유닛은 조합식을 재귀 추적해 자동 계산됩니다.
③ 계보 & 조합 트리 — TREE 패널
트리 보기유닛 선택 시 우측에 조합 계보가 트리로 표시됩니다. 노드 클릭으로 하위 재료를 펼칩니다.
모두 펼치기카드 우측 상단 버튼으로 전체 계보를 한 번에 펼치거나 접습니다.
상위 조합단일 선택 시 해당 유닛을 재료로 쓰는 상위 히든/슈퍼히든이 상단에 표시됩니다.
조건 / 수량노드의 (조건)은 특수 조건, ×수량은 필요 개수입니다.
④ 기타
초기화헤더 초기화 버튼으로 모든 선택 해제. 선택 정보는 자동 저장되어 새로고침 후에도 유지됩니다.
모바일세로 모드: 코스트→도감→트리 순 세로 배치. 가로 모드: PC와 동일한 3열 배치.
