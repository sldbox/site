개복디 넥서스 사용법을 단계별로 안내합니다

1
종족 탭 선택 좌측 상단의 탭에서 내 유닛 종족을 고릅니다
테바 테메 토바 토메 저그 중립 혼종
▼
2
유닛 카드 클릭 원하는 유닛 카드를 클릭하면 코스트 대시보드에 재료가 자동 계산됩니다
▼
3
체크리스트 확인 우측 상단 ◧ 체크리스트 전환 버튼을 눌러 조합 순서와 재료를 확인하세요

🔍 검색창 이용 방법

[레전드] · [히든] · [슈퍼히든] 등급 유닛만 검색됩니다. 매직~헬 등급 하위 재료는 도감 탭에서 직접 클릭하세요.

단일 입력 타이커스 → 타이커스핀들레이 1마리 등록
수량 지정 타이커스*8 → 타이커스핀들레이 8마리 등록
다중 입력 타이커스*8/아몬 → 두 유닛 동시 등록

⚠ 슈퍼히든 전체 · 아몬 · 데하카 · 유물은 *8을 입력해도 최대 1마리로 고정됩니다.

🛒
장바구니 모드 내가 선택한 유닛들만 한 화면에 모아봅니다. 목표가 많을 때 유용해요.
✔
탭 전체 선택 현재 탭의 모든 유닛을 한 번에 선택하거나 해제합니다. 장바구니 모드 중엔 비활성화.
⟲
선택 초기화 등록된 모든 목표 유닛과 조립 완료 기록을 전부 지웁니다.

⚡ 수량 빠르게 조절하기

유닛 카드의 − / + 버튼을 길게 누르고 있으면 숫자가 점점 빠르게 바뀝니다. 여러 번 누를 필요 없어요!

📋 조합법 미리보기

유닛 카드의 재료 이름 부분을 클릭하면 팝업 툴팁으로 전체 조합법을 바로 볼 수 있습니다.

우측 상단 ◧ 체크리스트 전환을 눌러 재료 트리를 확인하고, 인게임에서 완료한 재료를 체크해 코스트를 차감하세요.

✔
전체완료 해당 재료를 한 번에 전부 완성 처리합니다. 코스트가 즉시 줄어듭니다.
＋
부분 완료 (1개 / 5개) 자동포탑처럼 많이 필요한 재료는 나눠서 부분 완료를 누를 수 있어요.

✨ 연관 재료 역추적 (하이라이트)

체크리스트에서 유닛 카드 이름 부분을 클릭하면 그 재료와 연결된 카드들이 밝게 빛납니다. 어떤 유닛을 위한 재료인지 한눈에 파악할 수 있어요.
