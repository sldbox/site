개복디 넥서스 가이드

유닛 도감 데이터베이스 (좌측 패널)

  • 종족별 탭을 눌러 해당 카테고리의 모든 유닛 조합법을 확인합니다. 탭 하단의 숫자 (선택수/전체수) 로 종족별 선택 현황을 쉽게 파악할 수 있습니다.
  • 목록은 총 필요 비용(Cost)이 비싼 순서로 자동 정렬되어 노출됩니다. (슈퍼히든 등급은 예외로 최상단 고정)
  • 유닛 카드를 클릭하면 즉시 선택되며, 우측 패널의 코스트 계산기에 데이터가 반영됩니다.
  • 상단의 [전체선택/해제] 버튼을 통해 해당 탭의 모든 유닛을 일괄 조작할 수 있습니다.

계보 시뮬레이션 (도감 내 통합)

  • 우측 상단의 통합 `[계보]` 버튼을 누르면 현재 선택된 유닛들 하단으로 전체 계보 트리가 한 번에 상세히 펼쳐집니다. 계보 버튼을 끄면 트리가 즉시 숨겨집니다.
  • 모두 펼치기/접기 버튼으로 복잡한 하위 재료들을 한눈에 파악할 수 있습니다.

일일 정수 카운터 (자동 병합 시스템)

  • 도감에서 선택한 유닛은 자동으로 우측 매트릭스의 정수 카운터(총정수/코랄/아이어/제루스)에 실시간 합산 반영됩니다.
  • 선택된 최상위 유닛의 조합식을 밑단까지 스캔하여 모든 하위 히든/슈퍼히든 유닛을 추적하며, 중복되는 레시피라도 고유 유닛 1개당 1코스트로 자동 병합 처리하여 정밀하게 총합 계산됩니다.
  • 선택된 유닛의 컨트롤 영역에는 [정수획득] 태그가 명확하게 표시됩니다.

총합 코스트 계산 매트릭스 (우측 패널)

  • 선택한 최종 유닛들을 제작하기 위해 필요한 가장 기초적인 매직 등급 유닛의 총 요구 개수를 역산하여 한눈에 보여줍니다.
  • 수량을 늘리거나 줄일 때마다 모든 항목이 즉각적으로 자동 계산됩니다.
