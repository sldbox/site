🎮 개복디 넥서스 V2.0 패치노트


1. 디자인: 눈이 편안한 '올블랙' 변신
•	검은색 테마 적용: 배경을 완전한 검은색으로 바꿔서 오래 봐도 눈이 아프지 않아요.
•	중요 정보 강조: 꼭 필요한 글자와 버튼에만 색깔을 넣어서, 복잡한 게임 중에도 정보를 한눈에 찾을 수 있게 디자인을 다듬었습니다.

2. PC 버전: 답답함 없는 '풀 와이드' 화면
•	넓게 쓰는 화면: 양옆의 빈 공간을 없애고 화면을 꽉 차게 만들었습니다. 덕분에 유닛 목록과 조합법을 훨씬 시원하게 볼 수 있습니다.
•	따로따로 스크롤: 이제 왼쪽 메뉴를 내린다고 오른쪽 화면이 같이 움직이지 않아요. 내가 보고 싶은 칸만 골라서 편하게 내릴 수 있습니다.

3. 모바일 버전: 한 손으로도 척척! '스마트 레이아웃'
•	중요한 것부터 먼저: 화면이 좁은 모바일 특성에 맞춰 [계산기] → [조합법] → [유닛 족보] 순서로 가장 필요한 기능부터 먼저 보이게 바꿨습니다.
•	자동 화면 이동: 유닛 이름을 누르면, 상세 정보가 나오는 곳으로 화면이 알아서 '슝-' 하고 이동합니다. 손가락으로 한참 내릴 필요가 없어요.
•	숫자 칸 고정: 복잡한 숫자 칸들이 모바일에서도 깨지지 않고 예쁜 정렬(5x4)을 유지하도록 만들었습니다.

4. 일일정수: 보기 편해진 '요약판'
•	점수 한눈에 보기: 화면 위쪽에 코랄, 아이어, 제루스 점수판을 가로로 나란히 배치해서 지금 내 상태를 바로 알 수 있습니다.
•	접고 펼치기 기능: 목록이 너무 길어서 불편하셨죠? 이제 종족 이름을 누르면 목록이 나타나고, 다시 누르면 쏙 들어가는 기능을 넣었습니다.

5. 검색 필터: 색깔로 찾는 '등급 검색'
•	등급별 색상 적용: 유닛 이름에 등급 색깔(레전드는 보라색, 히든은 초록색 등)을 입혔습니다. 이제 이름만 봐도 얼마나 좋은 유닛인지 바로 알 수 있어요.
•	번호로 순서 정리: 유닛들이 세로로 번호가 매겨져 있어 목록을 읽기가 훨씬 쉬워졌습니다.

6. 더 빨라진 반응 속도
•	버벅임 감소: 구형 휴대폰이나 느린 PC에서도 화면이 끊기지 않고 부드럽게 작동하도록 내부 기능을 튼튼하게 보강했습니다.
